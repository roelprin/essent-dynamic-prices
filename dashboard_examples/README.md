# Dashboard Edition

Deze map bevat dashboardvoorbeelden voor Essent Dynamic Prices.

## Bestanden

- `dashboard_edition.yaml`  
  Complete verticale dashboardkaart met tegels, dagoverzicht, grafiek en slimme samenvatting.

- `dashboard_48h.yaml`  
  Losse ApexCharts-kaart voor vandaag + morgen.

## Vereist

Installeer via HACS:

- ApexCharts Card

## Entity IDs

De voorbeelden gebruiken:

```yaml
sensor.essent_dynamic_prices_uurprijzen
```

Als jouw entity anders heet, vervang die in de YAML.
